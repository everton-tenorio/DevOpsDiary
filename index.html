$ how_it_works

Each level is a sequence of tasks that build on each other. The order matters - task 5 assumes you completed tasks 1 through 4. That progression is intentional.

There is no required format. Use any environment: local VM, cloud free tier, or your current work setup. The goal is that you actually do the work and understand what you built.

Start at Beginner even if you have some experience. The early tasks build the foundation everything else depends on.
